Introduction

The Spike Sorter application allow you to execute 2 different applications offered by the KlustaSuite.
These applications are:

  • SpikeDetekt for detecting and extracting spikes from raw recordings.
    (klusta --detect-only)
  • KlustaKwik for automatically clustering the detected spikes.
    (klusta --cluster-only)

KlustaSuite is an open source spike sorting software suite designed for extracellular recordings obtained with large multielelectrode arrays (with hundreds of channels). It is developed by Cyrille Rossant, Shabnam Kadir, Dan Goodman, Max Hunter from the CortexLab at University College London, directed by Kenneth Harris and Matteo Carandini.

Usage

A brief explanation for each field on the left panel follows.

  1. Params file: is a text file (written in Python) containing all parameters that KlustaSuite requires to spike sort the data.
  2. Prb file: is a text file (written in Python) containing information about the probe.
  3. Step 1: Run detect: run the command line "klusta --detect-only".
  4. Step 2: Run cluster: run the command line "klusta --cluster-only".
  5. Raw data file: Spikes raw data file.
  6. Previous result file: is the zip file from a previous run, used to run the "Step 2: Run cluster".
  7. Result file: is the output zip file, containing all the generated files.
